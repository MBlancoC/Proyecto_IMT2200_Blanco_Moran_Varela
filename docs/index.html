1.  Objetivos y métodos del proyecto

Nuestro objetivo principal era entrenar un modelo de regresion lineal que nos ayude a predecir notas, nuestro objetivo secundario era responder algunas preguntas, como por ejemplo si habia alguna relacion entre el sexo y las notas del alumno. Para hacer todo esto, primero nos conseguimos un dataset de Kaggle, luego lo limpiamos y ajustamos ciertas variables para hacer su análisis más sencillo, también hicimos gráficos con las variables para visualizarlas mejor, para finalmente entrenar el modelo.
